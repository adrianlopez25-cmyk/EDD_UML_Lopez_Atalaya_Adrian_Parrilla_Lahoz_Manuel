## ¿Que es un UML?
UML es una notación estándar para métodos orientados a objetos, que fue creada mediante la unificación de las Técnicas de Modelado de Objetos. Se utiliza sobre todo para el analisis, diseño y despliegue de los sistemas. Esto permite la automatizacion del sofware y un mejor funcionamiento entre si.
## Beneficios de UML
- Nos permite describir de forma mas visual un programa. Es empleada sobre todo para esquematizar programas.
- Nos muestra las relaciones entre las clases y entidades del programa que vamos a desarrollar.
- Permite al programador reutilizar partes de código que ya existen en lugar de volver a escribir funciones ya que puede que ya se haya realizado programas similares previamente.
